hexo的主题配置和内容美化

有兴趣的可以用浏览器打开我的博客查看效果

主题配置

首先是主题的配置,每个主题的配置方式不同,我选择的是最简洁也是我比较喜欢的next主题,你也可以点击这里选择你喜欢的主题.

我们先进入你的博客项目所在的文件夹, 输入git clone https://github.com/theme-next/hexo-theme-next themes/next 下载next主题

下载好以后可以在themes目录下看到next主题的目录,在根项目的_config.yml搜索theme,主题默认应该是landscape 我们改成next,
在命令行中输入hexo server,启动成功后在浏览器打开http://localhost:4000 即可预览

打开 themes/next/_config.yml 搜索menu来配置菜单

这样首页就会有对应的归档,分类,相关,和标签啦,在协作的时候在文件头这么写

就可以指定主题和标签,单个可以直接在后面接入,多个的可以另起一行加 - 来添加

在文章中加入<!--more--> 可以实现查看更多

接下来我们搜索 Schemes 可以看到next提供的几种样式,可以选择自己喜欢的

我比较喜欢的是Gemini这种

接下来我们开启canvas背景,在配置文件中搜索canvas_nest将enable改为true即可开启

在全局的配置文件中搜索language改为zh-CN将全局的文字改为中文

hexo 设置看板娘

在博客文件夹主目录中执行npm install --save hexo-helper-live2d下载hexo看板娘支持

在这里 预览需要的看板娘,找到喜欢的看板娘后复制代码在目录中执行 npm install --save live2d-widget-model-hijiki 将你自己选择的看板娘代码替换成我选的这个,然后在博客根配置文件中添加

在use中修改成你选择对应看板娘的代码
